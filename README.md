- Funções interresantes para Strings
  - toLowerCase() → converte para minúsculo
  - toUpperCase() → converte para maiúsculo
  - trim() → remove espaços das bordas
  - substring(início) → corta a partir de uma posição
  - substring(início, fim) → corta um trecho específico
  - replace(char, char) → troca caracteres
  - replace(String, String) → troca palavras
  - charAt(posição) → pega um caractere
  - indexOf() → primeira ocorrência
  - astIndexOf() → última ocorrência
  - split() → divide a string em partes
